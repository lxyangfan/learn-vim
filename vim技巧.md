# 模式
normal模式
insert模式
visual模式

|命令|说明|
|---|---|
|<Esc>|从insert模式退回到normal模式|  
|`<C-[>`|同<Esc>|
|v|字符下,可视模式|  
|V|行模式下,可视模式|  
|<C-v>|块选择,可视模式|  

# 编辑

## 正向删除单词
例子：
The end is high

`daw` 可以使用d删除一个word, 并且它是可以重复的删除。即使用 `.`模式

## 插入模式下，及时删除

比如：
This is a line
可以使用 <Ctrl-w> 去向前删除一个 word, 并且当前不会退出insert状态。

## 切换输入模式和普通模式

<C-[> 和 <Esc> 切换输入模式和普通模式

## 替换
This is a line

`r` 和 `R` 可试一试 

# 窗口管理

## 创建窗口

|命令|说明|
|---|---|
|`<C-w>v`|纵向切分窗口 |  
|`<C-w>s`|横向切分窗口 |  
|`<C-w>q`|关闭窗口;    |
